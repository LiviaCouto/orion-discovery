## Sobre o produto

O Orion nasce para substituir a Sólides internamente na Listra Digital — uma plataforma de gestão de performance e recrutamento por aderência comportamental, construída com metodologia científica mais robusta, preço justo e foco total no que realmente gera valor.

> *"Da planilha para o produto. Do workaround para o sistema."*
